# RESTful API Node Server 
Install the dependencies:

```bash
yarn install
```

Set the environment variables:

```bash
cp .env.example .env

## Commands

Running locally:

```bash
yarn dev
```

Running in production:

```bash
yarn start
```

```Powershell
npm run start
```
